# HRI-Simulator-Assignment
This file is for a Human-Robot Interaction assignment. The purpose of the assignment is to build a single python file to interact with a robot in a simulator using hand gestures.

My projet intends to do the following:
  - ~~Point at objects to set queue them~~
  - ~~Use gesture to inform robot queue is complete~~
  - ~~robot stacks them in that order~~
  - Does not work simulator
  - new idea
  - 5 buttons on camera using bayes boxes
  - each do a specific action based on action
  - eg home, confirm, pick, place and cancel
